# Project Overview

This project applies cohort analysis to evaluate customer retention patterns in the e-commerce industry. Instead of analyzing all customers as a single group, the study segments users into monthly acquisition cohorts based on their first purchase date and tracks their behavior over time.

The goal is to understand:

How long customers remain active

How retention changes across different cohorts

The impact of time, seasonality, and marketing on customer loyalty

Which cohorts are most valuable to the business

By using a structured, data-driven approach, this project provides actionable insights that can help businesses improve customer engagement, reduce churn, and optimize marketing strategies.

# Objectives

This project aims to:

Segment customers into cohorts based on their first purchase month

Track customer retention over time using cohort aging

Compute retention rates across multiple time periods

Visualize retention trends using heatmaps and line charts

Provide insights that can guide business decisions such as:

Targeted marketing

Loyalty programs

Personalized recommendations

Customer lifetime value strategies
